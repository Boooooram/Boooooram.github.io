mysql - 更改root密码

发表于 2018-11-02 | 更新于 2019-04-26 | 分类于 mysql

查找mysql进程，查看mysql安装路径

ps -ef | grep mysql

进入mysql安装路径 /usr/local/Cellar/mysql/8.0.11/bin

cd /usr/local/Cellar/mysql/8.0.11/bin

暂停mysql服务

./mysql.server stop

安全模式启动mysql

./mysqld_safe --skip-grant-tables &

进入mysql交互模式

mysql

选择【mysql】数据库

use mysql

刷新mysql的系统权限相关表

flush privileges;

修改密码

ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';(8版本以前)
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '新密码';(8版本之后)

注意：重启mysql

./mysql.server stop
./mysql.server start
