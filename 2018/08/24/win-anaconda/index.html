刚安装好的conda需要更新一下

conda update -n base conda

熟悉 conda 的操作流程

创建一个名为 tensorflow 的环境

conda create --name tensorflow
或：conda create -n tensorflow

查询目前存在的所有环境(带*的表示当前所处环境)

conda info --envs
或：conda info -e

进入 tensorflow 环境

activate tensorflow

搜索要安装的包

conda search package_name

搜索包的目的：

  • 确认包是否存在(下图的 bs4 包就不存在，因此无法用法conda进行安装，此时可以选择用pip命令安装)
  • 列出该包所有的版本，并选择合适的版本安装

搜索 tensorflow 包

conda search tensorflow

安装 tensorflow

conda install tensorflow=1.9.0（强制安装指定的版本）
conda install tensorflow (安装最新的版本)


ps:

  • 安装包的时候会自动安装python3.6.6，如果你的当前环境的python版本不是3.6.6，系统就会强制升级or降级python

编写 tensorflow 的 hello world

python(进入python界面)
import tensorflow as tf
hello = tf.constant("hello world")
sess = tf.Session()
sess.run(hello)

如果对 tensorflow 不太了解的话，看一下 ➡️ 官方文档

退出 tensorflow 环境

deactivate

移除 tensorflow 环境

conda remove --name tensorflow --all
或：conda remove -n tensorflow --all

查询目前存在的所有环境(带*的表示当前所处环境)

conda info --envs
或：conda info -e
