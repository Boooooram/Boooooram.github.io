mysql-change-password

发表于 2018-11-02 | 分类于 mysql

查找mysql进程，查看mysql安装路径

ps -ef | grep mysql

进入mysql安装路径 /usr/local/Cellar/mysql/8.0.11/bin

cd /usr/local/Cellar/mysql/8.0.11/bin

暂停mysql服务

./mysql.server stop

安全模式启动mysql

./mysqld_safe --skip-grant-tables &

进入mysql交互模式

mysql

选择【mysql】数据库

use mysql

刷新MySQL的系统权限相关表

flush privileges;

修改密码

ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';

注意：重启mysql

./mysql.server stop
./mysql.server start

熟悉 conda 的操作流程

创建一个名为 tensorflow 的环境

conda create --name tensorflow
或：conda create -n tensorflow

查询目前存在的所有环境(带*的表示当前所处环境)

conda info --envs
或：conda info -e

进入 tensorflow 环境

activate tensorflow

搜索要安装的包

conda search package_name

搜索包的目的：

  • 确认包是否存在(下图的 bs4 包就不存在，因此无法用法conda进行安装，此时可以选择用pip命令安装)
  • 列出该包所有的版本，并选择合适的版本安装

搜索 tensorflow 包

conda search tensorflow

安装 tensorflow

conda install tensorflow=1.9.0（强制安装指定的版本）
conda install tensorflow (安装最新的版本)


ps:

  • 安装包的时候会自动安装python3.6.6，如果你的当前环境的python版本不是3.6.6，系统就会强制升级or降级python

sess = tf.Session()
sess.run(hello)

Anacoda常用命令

发表于 2018-05-31 | 更新于 2018-08-24 | 分类于 Anacoda

anacoda配置

anaconda换源：

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

恢复默认源

conda config --remove-key channels

有资源显示源地址：

conda config --set show_channel_urls yes

显示配置信息

conda config --show

anacoda使用

更新 conda 自身

conda update conda

更新 anaconda 自身

conda update anaconda

更新 python

conda update python

创建环境

conda create --name [name] [dependent package list]

例如：
conda create --name xxxxxxxx biopython
指定了python版本
conda create --name xxxxxxxx python=3.5 astroid babel

从其他环境拷贝到新的环境

conda create --name target --clone source

激活环境

Linux and macOS: source activate snowflakes
Windows: activate snowflakes

列出当前环境

conda info --envs

失效环境，或切换到root环境

Linux, macOS: source deactivate
Windows: deactivate

删除指定环境

conda remove --name xxxxxxxx --all

包管理命令

查询可用包版本信息

conda search --full-name python
conda search beautifulsoup4


查询在线包链接：
https://anaconda.org/

查询当前环境中的包列表

conda list

在环境中安装包

conda install numpy=1.13.3
conda install --name [环境名称] beautifulsoup4


使用pip工具安装:
pip只是一个包管理工具，不同于conda，是无法识别环境的，而且无法更新python，pip认为python不是一个包。但pip可以安装一些conda无法安装的包，pip已经随着anaconda和minconda安装好了。

安装命令
pip install see

删除包

conda remove --name [环境名称] iopro
