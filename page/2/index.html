java - 连接mysql但找不到驱动类

发表于 2018-05-10 | 更新于 2019-02-14 | 分类于 java

javaweb项目连接mysql的时候，出现了如下错误信息： 

java.lang.ClassNotFoundException: com.mysql.jdbc.Driver

把mysql的jar包添加到项目中

File - Project Structure - Project Settings - Libraries

点击”+”, 选择”Java”

选择mysql的jar包

选择项目

点击”Apply”&”OK”

把mysql的jar包扔到tomcat的lib目录

启动服务器


java - IDEA中,把java项目变成javaweb项目

发表于 2018-05-09 | 更新于 2019-02-14 | 分类于 java

新建项目




添加系统的tomcat服务

File - Settings - Build, Execution, Deployment - Application Servers

点击”+”, 选择”Tomcat Server”

选择tomcat安装路径

配置项目结构 - Modules

File - Project Structure - Modules

点击”+”, 选择”2 Libraries”

选择”Tomcat 9.0.7”, 点击”Add Selected”


配置项目结构 - Facets

File - Project Structure - Facets

点击”+”, 选择”Web”

选择”javaweb”(项目名称)

点击”OK”, 会跳到”Modules”

点击右下角的”Create Artifact”


修改”Name”为”javaweb:war exploded”(javaweb为项目名称)

点击”Apply”&”OK”后,项目根目录里出现”web”目录

添加项目的toncat服务

Run - Edit Configurations

点击”+”, 选择”Tomcat Server - Local”


修改”Name”为”Tomcat 9.0.7”(强迫症)

点击”Deployment”

点击”+”, 选择”Artifact”

点击”Apply”&”OK”

运行”HelloWorld”

在”src”目录中添加”HelloWorld.java”文件，并添加如下代码:

// 导入必需的 java 库
import java.io.*;
import javax.servlet.*;
import javax.servlet.http.*;

// 扩展 HttpServlet 类
public class HelloWorld extends HttpServlet {

    private String message;

    public void init() throws ServletException
    {
        // 执行必需的初始化
        message = "Hello World";
    }

    public void doGet(HttpServletRequest request,
                    HttpServletResponse response)
            throws ServletException, IOException
    {
        // 设置响应内容类型
        response.setContentType("text/html");

        // 实际的逻辑是在这里
        PrintWriter out = response.getWriter();
        out.println("<h1>" + message + "</h1>");
    }

    public void destroy()
    {
        // 什么也不做
    }
}

在 web - WEB-INF - web.xml 中添加如下代码:

<servlet>
    <servlet-name>HelloWorld</servlet-name>
    <servlet-class>HelloWorld</servlet-class>
</servlet>

<servlet-mapping>
    <servlet-name>HelloWorld</servlet-name>
    <url-pattern>/HelloWorld</url-pattern>
</servlet-mapping>

运行tomcat
