1. 安装显卡驱动

1.1 一些命令

sudo apt update
sudo apt install -y vim

1.2 禁用nouveau

1.2.1 将nouveau加入黑名单

sudo vim /etc/modprobe.d/blacklist.conf

1.2.2 添加在最后一行，禁用nouveau第三方驱动，之后也不需要改回来

blacklist nouveau

1.2.3 重新生成initram

sudo update-initramfs -u

1.2.4 重启，用该命令检查是否禁用成功；如果禁用成功，该命令无任何输出；否则就是禁用失败

lsmod | grep nouveau

1.3 进入纯终端界面

Ctrl + Alt + F1

1.4 关闭X服务

sudo service lightdm stop

小提示：在命令行输入：sudo service lightdm start ，然后按Ctrl-Alt+F7即可恢复到图形界面。

1.5 卸载原有驱动

sudo apt remove --purge nvidia*

1.6 安装驱动

1.6.1 自己下载驱动

下载地址: https://www.nvidia.cn/Download/Find.aspx

cd ~/Download
chmod +x ./NVIDIA-Linux-x86_64-384.59.run
sudo ./NVIDIA-Linux-x86_64-384.59.run --no-opengl-files

–no-opengl-files：表示只安装驱动文件，不安装OpenGL文件。这个参数不可省略，否则会导致登陆界面死循环，英语一般称为”login loop”或者”stuck in login”。

必选参数解释：因为NVIDIA的驱动默认会安装OpenGL，而Ubuntu的内核本身也有OpenGL、且与GUI显示息息相关，一旦NVIDIA的驱动覆写了OpenGL，在GUI需要动态链接OpenGL库的时候就引起问题。

1.6.2 apt安装驱动（但不一定下得到最新）

sudo atp install -y nvidia-

然后猛击tab键两次（也可以轻点），下面就会出来许多版本的驱动，当然是安装一个版本最高的，例如

sudo atp install -y nvidia-384

1.6.3 重启查看安装情况

sudo reboot

重启后打开终端，输入 

nvidia-smi

如果安装失败，重新从第 1.3 步开始走一遍

2. 安装CUDA

2.1 根据自己显卡和操作系统的版本下载对应的CUDA版本

地址: https://developer.nvidia.com/cuda-toolkit-archive

2.2 进入纯终端界面

Ctrl + Alt + F1

2.3 关闭X服务

sudo service lightdm stop

2.4 安装

cd ~/Download
chmod +x ./cuda_8.0.61_375.26_linux.run
sudo ./cuda_8.0.61_375.26_linux.run --no-opengl-libs
  • –no-opengl-libs：表示只安装驱动文件，不安装OpenGL文件。必需参数，原因同上。注意：不是-no-opengl-files。
  • –uninstall (deprecated)：用于卸载CUDA Driver（已废弃）。
  • –toolkit：表示只安装CUDA Toolkit，不安装Driver和Samples。
  • –help：查看更多高级选项。

    Not enough space on parition mounted at /tmp.Need 5091561472 bytes.Disk space check has failed. Installation cannot continue.

即错误提示为/tmp空间不足，可执行下面的操作：

sudo mkdir /opt/tmp         
sudo ./cuda_8.0.61_375.26_linux.run --no-opengl-libs --tmpdir=/opt/tmp/

在根目录下的opt文件夹中新建tmp文件夹，用作安装文件的临时文件夹

2.5 测试

2.5.1 编译并测试设备 deviceQuery：

cd /usr/local/cuda-8.0/samples/1_Utilities/deviceQuery
sudo make
./deviceQuery

2.5.2 编译并测试带宽 bandwidthTest：

cd ../bandwidthTest
sudo make
./bandwidthTest

如果这两个测试的最后结果都是Result = PASS，说明CUDA安装成功啦。

2.6 安装补丁

cd ~/Download
chmod +x ./cuda_8.0.61.2_linux.run
sudo ./cuda_8.0.61.2_linux.run
  • 最后安装补丁CUDA官方补丁

3. 安装cuDNN

下载好cudnn文件后执行以下几步： 

tar -xzvf cudnn-8.0-linux-x64-v6.0.tgz
sudo cp -a cuda/include/cudnn.h /usr/local/cuda/include
sudo cp -a cuda/lib64/libcudnn* /usr/local/cuda/lib64
